## Setup:
### Steps to run:
    python -m pipenv shell
    pip install -r requirement.txt
    uvicorn main:app --reload
    
    Hit localhost:8000/docs in the browser to run Swagger to view API's


### Command for virtual env:

    python -m pipenv shell
    exit

### Dependencies:
    bcrypt
    PyJWT
    uvicorn
    SQLAlchemy
    fastapi

    install SQLLite for database
